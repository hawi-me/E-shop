#  E-Shop

A modern **eCommerce web app** built with **Next.js (App Router)** and **TypeScript**.  
It displays products fetched from [DummyJSON API](https://dummyjson.com), supports CRUD operations, search, pagination, and favorites using **Redux Toolkit**.

---

##  Project Overview

**Objective:**  
Build a simple eCommerce page using modern frontend technologies to practice real-world skills.

**Key Features:**
- Product listing with pagination (infinite scroll)
- Product details view
- Add/remove favorites using Redux Toolkit
- Create, edit, and delete products (CRUD)
- Search products by keyword
- Responsive design with Tailwind & Shadcn UI
- Toast notifications for feedback
- Dark mode toggle (via Redux)
- Bonus: Basic mock authentication (optional)

---

##  Tech Stack

| Category | Technology |
|-----------|-------------|
| Framework | [Next.js 14 (App Router)](https://nextjs.org/) |
| Language | [TypeScript](https://www.typescriptlang.org/) |
| Styling | [Tailwind CSS](https://tailwindcss.com/) |
| UI Library | [Shadcn UI](https://ui.shadcn.com/) |
| State Management | [Redux Toolkit](https://redux-toolkit.js.org/) |
| API | [DummyJSON](https://dummyjson.com/docs/products) |
| HTTP Client | [Axios](https://axios-http.com/) |

##  Installation & Setup

###  Clone the Repository
```bash
git clone https://github.com/hawi-me/E-shop
cd E-shop
npm install
npm run dev
```
